# E-commerce Customer Analytics System

A comprehensive, production-ready Customer Analytics system designed to process transaction logs, perform RFM segmentation, predict future customer churn risk using machine learning, project 12-month Customer Lifetime Value (CLV), and serve insights through a high-fidelity interactive dashboard.

---

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have Python 3 installed. The pipeline uses standard data science libraries. If not already installed, run:
```bash
pip install pandas numpy scikit-learn matplotlib seaborn
```

### 2. Running the End-to-End Pipeline
To execute the data generator, cleaner, machine learning modeling, visualization scripts, and compile the dashboard dataset, run:
```bash
python src/run_pipeline.py
```
This automatically produces all deliverables in the workspace:
*   `data/raw_transactions.csv`: Simulated transaction logs (325k lines).
*   `data/cleaned_transactions.csv`: Preprocessed transactions.
*   `data/customer_metrics.csv`: Customer analytics profiles (RFM, CLV, Churn Probabilities).
*   `data/plots/`: Production-ready charts (PNG format).
*   `dashboard/data.js`: Unified analytical payload for the UI.

### 3. Running SQL Queries
To verify and extract analytical insights using the SQL script (with SQLite adaptation), run:
```bash
python src/verify_sql.py
```
This query set executes standard ANSI business-intelligence queries and outputs the results to:
*   `data/sql_insights_report.txt`

### 4. Viewing the Executive Dashboard
Double-click the HTML file to open the dashboard locally in any web browser:
*   `dashboard/index.html`

*No local server or database connection is required; the dashboard loads compiled metrics directly from `data.js` without CORS security restrictions.*

## 📊 Core Analytics Summary

### 1. Cohort Analysis & Loyalty
*   **Total Customer Count**: 3,000 unique clients over a 24-month horizon.
*   **Repeat Purchase Rate (RPR)**: **79.53%**, showcasing extremely strong customer retention and product-market fit.
*   **Seasonality**: Holiday campaigns (November/December) yield over **80% MoM revenue spikes**, followed by a typical post-holiday correction in January.

### 2. RFM Revenue Concentration (80/20 Pareto)
Analysis reveals a stark concentration of revenue, confirming Pareto principles:
*   **Champions** (24.7% of customers) contribute **78.46% of revenue** ($9.62M).
*   **Loyal Customers** (19.2% of customers) contribute **12.90% of revenue** ($1.58M).
*   *Retention Recommendation*: 91.36% of sales are generated by the top ~44% of customers. Loyalty rewards and VIP programs for Champions/Loyalists should receive the highest budget allocation.

### 3. Churn Prediction & Win-Back Alerts
The Random Forest model trained on behavioral feature subsets (accuracy: **93.79%**) identified **74 high-value VIP customers** who have drifted into the "Can't Lose Them" cohort (R=1, F>=3, M>=3) and show a churn probability > 50%:
*   *Top Risk Case*: Customer #10452 has spent $2,860.21, but has been inactive for **419 days** (94.05% churn probability).
*   *Action Plan*: High-value win-back campaigns, telephone interviews, or executive outreach should be targeted directly at these 74 accounts.

## 🎨 Executive Dashboard Interface

The custom dashboard utilizes a sleek, dark glassmorphism layout tailored for corporate leadership:
*   **Live KPI Panel**: Displays Revenue, Customer count, AOV, Churn risk, and Average CLV.
*   **Visual Charts**: Rendered using Chart.js, featuring responsive grid ticks and dual-axis alignment.
*   **CRM Recommendation Deck**: Selecting any customer segment dynamically updates the panel with targeted characterizations and step-by-step retention actions.
*   **Re-engagement Panel**: Displays a priority win-back table of at-risk VIPs with direct call-to-action buttons.
